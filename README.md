Orienteering
================

Rails project for Orienteering



Ruby on Rails
-------------

This application requires:

- Ruby 2.1.3
- Rails 4.1.6


Travis CI
-------------

- https://travis-ci.org/neutrino/orienteering_app


Server deployment
================

Server needs:

* Ruby 2.1.3
* PostgreSQL >= 8.4
* Nginx (production use)

Other packages that might be needed (if not installed already):

* Openssl
* ImageMagick
